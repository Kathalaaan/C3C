Sticky Notes

Create draggable study notes, change their colors, and keep them saved in this browser.

  • Click `+ New sticky` to create a note.
  • Drag a note from its top bar to reposition it.
  • Use `+` to duplicate, `🎨` to change color, and `🗑` to delete.
  • Your note text, color, and position are saved in local storage.

Quantitative Aptitude Folder

This folder keeps your study materials together on this browser. Upload files, save useful links, and write text notes for revision.

  • Use `Upload files` to store PDFs, PNG, JPG, docs, spreadsheets, or any other study file in this browser folder.
  • Paste useful URLs in the link form and save your own explanations in the note form.
  • Use `Open` to view links, `Download` for saved files, and `Delete` to remove an item.
  • Everything here stays tied to this browser storage, so use the same browser on the same device to keep access.
